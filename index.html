Uso del padding, border y margin:

Padding:

Los padding se utilizan para generar espacio alrededor del contenido de un elemento, dentro de cualquier borde definido.

Ejemplo:

En este espacio estamos usando el padding, ya que le estamos generando un espacio alrededor de nuestro texto.
Todos los lados con la misma medida.



Border:

Las propiedades de borde CSS permiten especificar el estilo, ancho, color del borde de un elemento.

Ejemplo:

Un borde punteado.

Un borde discontinuo.

Un borde solido.



Margin:

Los margin se utilizan para crear un espacio alrededor de los elementos, fuera de cualquier borde definido.

Ejemplo:

En este espacio se esta haciendo uso del margin, le estamos dando una distancia de 25px en todo su alrededor.
A diferencia del padding que lo que hace es dar una distancia dentro de un elemento contenedor, el margin lo que hace es separar la distancia indicada de todos los elementos.


Diferencias entre elementos inline y elementos bloque:

Inline:

- Son aquellos elementos que solo ocupan el espacio necesario y no generan un salto de línea.

- Se usan dentro de otros elementos y pueden estar en la misma línea.

- Ejemplos: span, a, strong, em, img, br, label, input.

Block:

- Son aquellos elementos que siempre comienzan en una nueva linea.

- Siempre ocupan todo el ancho disponible (Se extiende hacia la izquierda y la derecha tanto como puede).

- Ejemplos: p, div, address, dd, header, h1-h6. entre otros
