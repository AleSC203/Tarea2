Inline:

- Se usan dentro de otros elementos y pueden estar en la misma línea.

- Ejemplos: span, a, strong, em, img, br, label, input.

Position: Static

¿Como funciona?
Los elementos HTML se posicionan de forma estática de forma predeterminada.
un elemento con posicion estática no se posiciona de ninguna manera especial,
ya que siempre se posiciona de acuerdo al flujo normal de la página.

Uso de la posición statica

Position: relative

¿Como funciona?
El elemento sigue ocupando su espacio habitual en el documento, como sino tuviera posición absoluta o fija
Es util cuando se desea mover un elemento sin afectar el diseño de los demás,
ya que su despazamiento no altera el flujo del documento.

Uso de la posición relativa

Position: absolute

¿Como funciona?
La posición absoluta en CSS es un método de posicionamiento que permite que un elemento se ubique en la página independientemente del flujo normal del documento.

Uso de la posición absoluta

Position: fixed

¿Como funciona?
Se posiciona con respecto a la ventana gráfica, lo que significa que permanece en la misma posición incluso al desplazarse por la página.
Las propiedades top, right, bottom y left se utilizan para posicionar el elemento.
El ejemplos para esta propiedad lo puedes ver en un div con borde verde ubicado en la parte derecha inferior de la pantalla .

Uso de Float none y Clear none

La propiedad float: none; indica que el elemento no flotará ni a la izquierda ni a la derecha. Esto significa que la imagen mantendrá su posición dentro del flujo natural del documento, sin que otros elementos la rodeen.

La propiedad clear: none; indica que este elemento no aplicará ninguna limpieza de flotantes. Esto significa que seguirá colocándose junto a elementos flotantes si los hay, sin forzar un salto de línea.



Uso de Float both y Clear both

La propiedad float: both; no es comúnmente utilizada, pero en algunos navegadores puede interpretarse como una flotación tanto a la izquierda como a la derecha, aunque su comportamiento puede ser impredecible. Generalmente, se recomienda usar `left` o `right` en lugar de `both`.

La propiedad clear: both; asegura que este elemento no quede junto a ningún elemento flotante, ya sea a la izquierda o derecha. Se usa comúnmente después de bloques flotantes para evitar que el diseño se desajuste.



Exploración de la propiedad Display en CSS

Display: block;

¿Qué es?
Este valor hace que el elemento ocupe todo el ancho disponible y siempre comience en una nueva línea.

¿Para qué se usa?
Se usa en elementos como `div`, `p`, `section`, permitiendo estructurar el contenido en bloques separados.

Ejemplo:

Hola

hi

Nota: A simple vista no se nota pero hola y hi se encuentran dentro de una etiqueta p la cual hace que no puedan estar ambas palabras en la misma linea.

Display: inline;

¿Qué es?
Este valor permite que el elemento se mantenga en la misma línea con otros elementos.

¿Para qué se usa?
Se usa para estilos de texto como `span`, `a`, `strong`, lo que ayuda a aplicar estilos sin afectar la disposición.

Ejemplo:

Hola, hi

Nota: En este ejemplo ambas palabras se encuentran en etiquetas de span distintas pero esta etiqueta es span la cual si permite que sigan en la misma linea.

Display: inline-block;

¿Qué es?
Similar a `inline`, pero permite modificar el tamaño con `width` y `height`.

¿Para qué se usa?
Se usa en botones, tarjetas y elementos que deben alinearse sin afectar el flujo normal.

Display: none;

¿Qué es?
Este valor oculta completamente el elemento, sin ocupar espacio en la página.

¿Para qué se usa?
Se usa para menús ocultos, elementos dinámicos o contenido que se activa con JavaScript.

Uso del Overflow: visible

El contenido que excede el contenedor se muestra fuera de sus límites sin restricciones.
Este es el comportamiento que toma el overflow: visible Este es un ejemplo de prueba.

Uso del Overflow: hidden

El contenido que excede el contenedor se oculta y no es accesible.
Este es el comportamiento que toma el overflow: hidden Útil para evitar desbordamientos no deseados en layouts específicos.

Uso del Overflow: scroll

Se agregan barras de desplazamiento (scroll) tanto en eje X como en eje Y si el contenido supera el tamaño del contenedor.
Este es el comportamiento que toma el overflow: scroll.
Ideal para contenedores con contenido grande que debe ser accesible mediante desplazamiento.

Uso del Overflow: auto

Se agrega scroll solo cuando es necesario (similar a scroll, pero sin forzar siempre la barra).
Este es el comportamiento que toma el overflow: auto.
Recomendado en la mayoría de los casos para manejar contenido variable sin forzar el scroll cuando no es necesario.

Z-index

¿Para que funciona?
El Z-index en CSS controla la posición en la profundidad de los elementos en una página web,
es decir, determina qué elemento aparecerá delante o detrás cuando se superponen.


Algunas de sus propiedades son:

myBox
z-index 0
z-index 1
z-index 2
z-index 3
Prueba auto
Prueba inherit
Prueba initial
Prueba unset

Breve explicación de cada propiedad de `z-index`:
Auto: Es el valor por defecto y sigue el orden normal de apilamiento.
inherit: Hereda el valor de `z-index` del padre.
initial: Restablece el `z-index` a su valor predeterminado (`auto`).
unset: Se comporta como `inherit` si el padre tiene `z-index`, y como `initial` si no lo tiene.
